## Getting started

### Prerequisites
- Node.js 20+
- pnpm 8+
- Python 3.11+
- Docker (optional, for local infra)

### Install
git clone git@github.com:YOUR_ORG/kofoundr.git
cd kofoundr
cp .env.example .env
pnpm install

### Run (development)
pnpm dev
